Projects

COVID-19 Prediction Using Machine Learning

This project is an advanced COVID-19 prediction system built using machine learning techniques, designed to analyze the progression and impact of the pandemic. It utilizes real-world datasets to provide insights into infection trends, recovery rates, and fatality patterns, offering valuable data for decision-making and resource allocation. The system features predictive modeling capabilities, enabling accurate forecasting of case counts and potential hotspots, along with visual analytics for a comprehensive understanding of the pandemic's trajectory. The models were developed using Python and its extensive libraries, such as Pandas, NumPy, and Scikit-learn, ensuring efficient data processing and robust prediction mechanisms. Overall, this project showcases the transformative role of AI in public health, providing a scalable and actionable solution to assist in pandemic preparedness and management.

Sentiment Analysis Tool

This sentiment analysis tool is built using Python and NLTK, designed to classify user-provided text as positive, negative, or neutral. It features a user-friendly GUI developed with Tkinter, allowing for easy text input and immediate analysis. The project employs advanced NLP techniques to ensure accurate sentiment classification. Additionally, it integrates Matplotlib to visualize the sentiment distribution, helping users better understand the results. Overall, this tool enhances user engagement and provides valuable insights into the emotional tone of the text.

RFID Attendance System

This contactless attendance system utilizes RFID technology to streamline attendance tracking, incorporating essential components such as the RFID Reader, RFID Card, NodeMCU ESP8266, and an LCD display with I2C. The system is programmed with Arduino code for real-time attendance monitoring, demonstrating improved skills in embedded systems and hardware integration. Utilizing a breadboard and jumper wires, the project provides a practical solution for efficient attendance management in various environments. Overall, this innovative setup enhances the traditional attendance process by leveraging modern technology for seamless tracking.

Smart Parking System

This project is a comprehensive parking management system developed using Java with the Spring Boot framework and H2 Database, enabling effective management of parking spaces. It features real-time availability tracking, allowing users to check the status of parking spots at any given moment, as well as booking capabilities that streamline the reservation process. Users receive timely notifications regarding their bookings, ensuring they stay informed throughout their experience. The frontend of the application is crafted with HTML, CSS, and JavaScript, reflecting my skills in creating intuitive and responsive web interfaces. Overall, this project not only enhances user experience but also significantly improves operational efficiency by automating and simplifying the parking management process.

Arduino Obstacle Avoiding Robot

This DIY Arduino robot is designed with advanced functionalities, including obstacle avoidance, voice control, and Bluetooth control, showcasing a robust understanding of robotics and control systems. The robot utilizes ultrasonic sensors for real-time obstacle detection and avoidance, ensuring safe navigation through its environment. Voice control is integrated through a speech recognition module, allowing for intuitive commands and enhanced user interaction. Additionally, Bluetooth control enables remote operation via a smartphone app, providing versatile control options. This project has significantly enhanced my skills in hardware integration and programming, emphasizing the critical role these elements play in developing autonomous robotic systems.
