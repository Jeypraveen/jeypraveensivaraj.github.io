EduSense AI - Comprehensive Educational Platform

EduSense AI is an advanced educational platform that leverages cutting-edge AI to transform hybrid and remote learning. It integrates three powerful modules: automated attendance tracking, real-time emotion and cognitive state analysis, and intelligent exam proctoring, all powered by a unified FaceNet-based face recognition pipeline. Using MTCNN and InceptionResnetV1, the system captures and logs attendance with over 95% accuracy while preventing proxy entries. Emotion and attention are monitored using MediaPipe Face Mesh and a custom CNN to detect seven emotions and cognitive states like drowsiness or distraction in real time. During exams, YOLOv8 identifies unauthorized devices and suspicious activity, generating timestamped violation reports to ensure academic integrity. Designed with a modular architecture and built using technologies like TensorFlow, OpenCV, and PyTorch, EduSense AI runs efficiently on standard hardware. It improves student engagement, reduces administrative workload, supports mental well-being, and offers a scalable solution suitable for both hybrid and remote educational environments.

Hybrid Facial Analysis System for Emotion and Cognitive State Recognition

Designed and developed a hybrid facial analysis system that integrates state-of-the-art deep learning via convolutional neural networks (CNNs) with classical computer vision techniques to classify seven core human emotions (happiness, sadness, anger, surprise, fear, disgust, neutrality) and three cognitive states (attention, drowsiness, distraction) from facial video. Trained and validated on the Kaggle Facial Emotion Recognition dataset, the model achieved 84 percent overall accuracy across varied lighting conditions, head poses, and demographic differences. To meet diverse application needs, the system offers three analysis modes: a low-latency webcam mode for real-time emotion and attention monitoring; a single-video mode for detailed, frame-by-frame post-processing of individual files; and a folder-batch mode that automatically traverses and analyzes entire directories of videos for large-scale studies. Leveraging OpenCV for efficient frame capture, preprocessing, and feature extraction, this versatile framework can be deployed in advanced driver monitoring to detect fatigue and distraction, in e-learning platforms to gauge and adapt to student engagement, and in human–computer interfaces that respond dynamically to users’ emotional and cognitive cues. After analysis, the system generates a comprehensive report as a text file.

Brain Tumor Detection & Classification Using Deep Learning

Designed and implemented a comprehensive, end-to-end TensorFlow-Keras pipeline that seamlessly integrates U-Net-based image segmentation with convolutional neural network classification to detect and categorize brain tumors from MRI scans. After rigorous training and validation, the system achieved a classification accuracy exceeding 92 percent and a Dice coefficient above 0.88, ensuring both reliable identification and precise tumor boundary delineation. Advanced preprocessing routines built with OpenCV and NumPy automatically enhance contrast, remove artifacts, normalize intensity values, and resize scans for consistent model performance. The user interacts with the solution through an intuitive Gradio web interface that supports real-time tumor localization overlays on MRI slices, displays classification confidence scores, and provides quantitative measurements such as tumor volume and area. Upon completion of each analysis, the pipeline generates a comprehensive text report summarizing segmentation metrics, classification results, measurement statistics, and processing timestamps. This robust framework accelerates clinical workflows by offering transparent, reproducible, and actionable insights for diagnostic decision-making.

AI-Powered Python Code Performance Optimizer

Developed an AI-powered code optimization platform using Python, Gradio, and Google Gemini 1.5 Flash API that provides real-time static analysis, dynamic performance profiling, and intelligent optimization suggestions to assist developers in identifying and resolving performance issues efficiently. Gemini 1.5 Flash is a fast and cost‑effective multimodal model optimized for high‑volume, low‑latency tasks, making it well suited for live code optimization workflows. The platform parses code as it is edited to flag inefficient constructs, unused variables, and other static issues instantly. It then executes profiling runs to measure execution time and memory usage, pinpointing performance bottlenecks. When inefficiencies are detected, the system invokes Gemini 1.5 Flash to generate contextual suggestions such as algorithmic improvements, concurrency enhancements, vectorization techniques, or more efficient library alternatives. An intuitive Gradio interface allows developers to view static analysis warnings, inspect profiling metrics, and apply suggested optimizations with a single click. After each session, the system automatically produces a detailed text report summarizing static findings, performance metrics, applied optimizations, and timestamps.
